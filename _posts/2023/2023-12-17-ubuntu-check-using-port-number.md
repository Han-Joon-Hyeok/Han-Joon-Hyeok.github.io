# netstat

- netstat 을 사용하면 열려있는 포트 번호와 해당 포트를 사용하는 프로세스의 PID, 프로그램의 이름을 확인할 수 있다.

```bash
$ netstat -tulpn

 will not be shown, you would have to be root to see it all.)

...
```
